# Proyecto de Infraestructura: Autenticación Centralizada con LDAP y LIVA

![Estado del Proyecto](https://img.shields.io/badge/Estado-En_Desarrollo-green)

Este repositorio contiene la documentación y scripts de despliegue para el proyecto de fin de curso de 2º SMR.

## 📋 Descripción del Proyecto
Implementación de un sistema de **Directorio Activo (LDAP)** utilizando hardware de bajo consumo (**ECS LIVA**) para centralizar la gestión de usuarios, grupos y permisos en una PYME.
