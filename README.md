Repositorio-Trabajo-LDAP
Proyecto de Infraestructura: Autenticación Centralizada con LDAP y LIVA

Este repositorio contiene la documentación y scripts de despliegue para el proyecto de fin de curso de 2º SMR.

📋 Descripción del Proyecto

El objetivo es implementar un sistema de Directorio Activo (LDAP) utilizando hardware de bajo consumo (ECS LIVA) para centralizar la gestión de usuarios, grupos y permisos en una PYME. Se busca solucionar los problemas de seguridad y eficiencia derivados de la gestión de cuentas locales dispersas.

🛠️ Stack Tecnológico

Servidor: ECS LIVA (Ubuntu Server 22.04 LTS).

Directorio: OpenLDAP (slapd).

Seguridad: TLS/SSL y ACLs.

Cliente: libpam-ldap, nss-ldap.
